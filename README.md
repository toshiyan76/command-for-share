# command-for-share

開発効率を向上させるためのAIエージェント設定・コマンド定義集です。
現在は仕様駆動開発用のコマンドを定義しています。

## 概要

このリポジトリには、AIコーディングアシスタントや開発環境向けのカスタムコマンド、ワークフロー定義が含まれています。
これらの設定ファイルをあなたのプロジェクトに追加することで、開発プロセスを標準化し、効率化することができます。

## 対応ツール

以下のツールや環境で、定義されたコマンドやエージェント機能を利用できます：

- **AntiGravity**
- **Claude Code**
- **Codex**
- **Cursor**

## 使い方

このリポジトリに含まれるディレクトリ（`.agent`, `.claude`, `.codex` など）を、**あなたのプロジェクトのルートディレクトリに配置してください**。

配置後、各エディタやツールが設定を認識し、定義済みのコマンドやワークフローが利用可能になります。
